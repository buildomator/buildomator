<objective>
Close the two critical code-review findings (CR-01 broad hook-fallback stamp, CR-02
headless sanitizer) and the functional core of D-08: broaden the namespace rewrite from
the leading-slash-anchored `/\/gsd:/g` to `gsd:(?!/)` so the 69 `subagent_type`/`type`
agent references (`gsd:gsd-<agent>`) and the agent/skill frontmatter `name:` fields flip
to `bm:`. That is what actually makes a bm-only install spawn its own agents (BM-03).

verifier passed at the code level, but the reviewer found that agent references
(`gsd:gsd-executor`), 19 un-stamped hook-fallback carriers, and the SDK sanitizer all
still point at gsd, so `/bm:*` commands break or cross into the sibling plugin on a
bm-only install. This plan brings the transform up to D-08.

Output: an updated deterministic transform + build, regenerated `dist/bm`, and unit
coverage that proves the new scope. The fail-closed census, test-race, and hygiene fixes
are plan 13-04.
</objective>
