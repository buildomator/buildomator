| Boundary | Description |
|----------|-------------|
| committed dist/bm -> user installs | the marketplace source ./dist/bm ships whatever is committed at the release ref |
| CLAUDE_PLUGIN_ROOT -> hook execution | hook commands decide which cached plugin code runs on the user's machine |

| T-12-03 | Elevation of Privilege / Tampering | dist/bm hooks falling back to the gsd cache dir | mitigate (this phase) + defer (fix) | tripwire-fallback smoke assertion (Task 2) proves the primary CLAUDE_PLUGIN_ROOT path wins; verbatim fallback documented as a known limitation with the per-plugin fix filed for Phase 13/14 |
| T-12-04 | Information Disclosure | published package contents | mitigate | bm-build-drift job re-runs the drift test whose assertions forbid .git/.planning/.claude/node_modules/scratchpad in dist/bm |
| T-12-SC | Tampering | package installs | accept | no third-party packages introduced by this plan (workflow YAML + docs only) |
</threat_model>
